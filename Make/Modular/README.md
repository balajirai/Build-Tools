# Organized Project (Modular)

```swift
Modular/
│── Makefile
│── make/
│     ├── rules.mk
│     ├── sources.mk
│     └── variables.mk
│── header/
│     └── util.h
│── src/
│     ├── main.cpp
│     └── util.cpp
│── build/   (will be generated)
```

## Build
```bash
make
```

## Run
```bash
./app
```

## Clean
```bash
make clean
```
